## Overview

## Dataset
Source: [Netflix Dataset on Kaggle](https://www.kaggle.com/datasets/shivamb/netflix-shows)

## Project Files
| File Name | Description |
|------------|--------------|
| `Netflix_RawData.xlsx` | Original dataset downloaded from Kaggle (uncleaned) |
| `Netflix_CleanData.xlsx` | Cleaned version of the dataset with unnecessary columns removed |
| `Netflix_Top10Rated.xlsx` | Analysis of top 10 highest-rated Netflix titles based on IMDb ratings |
| `Netflix_Summary.xlsx` | Contains summary insights and final dashboard visuals |

## Objectives
1. What is the ratio of Movies vs TV Shows on Netflix?  
2. How has Netflix content changed over time?  
3. What are the top 10 highest-rated Netflix titles (based on IMDb)?

## Visuals
- Pie Chart: Ratio of Movies vs TV Shows  
- Line Chart: Netflix content growth over time  
- Bar Chart: Top 10 highest-rated titles  

*(Visualizations are available within the Excel files.)*

## Key Insights
- Netflix offers more Movies than TV Shows (approx.75.48%).  
- The number of titles has grown steadily since 2013.  
- Top-rated titles include: "Breaking Bad" (9.5), "Band of Brothers" (9.4), "The Shawshank Redemption" (9.3), "Avatar: The Last Airbender" (9.3), and "The Godfather" (9.2)
